EPI Memory

EPI Memory est un jeu qui permet de mémoriser le visage des étudiants à EPITECH !
Le principe est simple: vous avez une minute pour associer le maximum de login à des photos d'étudiants.
Combien en trouverez-vous ? Serez-vous assez rapide ?

Connexion

Ce jeu est réservé aux étudiants d'EPITECH, vous devez vous identifier pour pouvoir y accéder !
