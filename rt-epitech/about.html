EPITECH Raytracer gallery

This gallery is just a new interface for the old EPITECH raytracer's gallery.
It adds a more modern interface, an image gallery and the ability to order/filter images.

It is written in javascript using the awesome angularjs framework and a little bit of PHP to fetch the data from the legacy website. Indeed, the data is directly retrieved from the original interface so it reflects its content in real time.
